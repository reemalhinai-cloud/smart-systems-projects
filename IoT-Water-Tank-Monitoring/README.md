# IoT Water Tank Monitoring

## Project Description
This project monitors water levels in tanks using sensors and IoT technology.
It detects water leakage, monitors water levels, and alerts users about water shortages.
The system uses Arduino Mega and NodeMCU ESP8266 for automation and remote monitoring.

## Components
- Arduino Mega 2560
- NodeMCU ESP8266
- Ultrasonic sensor
- Water flow sensor
- Water sensor
- LCD display
- Relay module
