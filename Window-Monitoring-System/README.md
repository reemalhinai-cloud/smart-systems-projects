# Window Monitoring System

## Project Description
This project implements an IoT-based smart window monitoring and control system using NodeMCU. 
The system detects environmental conditions such as rain using sensors and automatically opens or closes the window. 
It also allows users to monitor and control the window remotely through IoT connectivity.

## Components
- NodeMCU ESP8266
- Water sensor
- Relay module
- DC motor
- Solar panel
- Battery
